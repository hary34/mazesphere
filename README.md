# MazeSphere

A simple Flask-based front-end demonstrating a whimsical maze-like playground. The page features a moving sphere inside a square maze.

## Running Locally

1. Ensure you have Python installed with `flask` available. If not, install it using:
   ```bash
   pip install flask
   ```
2. Start the development server:
   ```bash
   python app.py
   ```
3. Visit [http://localhost:5000](http://localhost:5000) in your browser.

Press **Start** to watch the ball wander around the maze.
